## 📖 Essential Commands

```bash
# Start Claude Code in current directory
claude

# Ask Claude to generate code
claude "Create a function to parse JSON config files"

# Review specific file
claude @src/main.py "Review this code for bugs"

# Explain code
claude @src/parser.py "Explain how this parsing logic works"

# Refactor
claude @src/old_code.py "Refactor this to use modern Python features"

# Debug
claude "Why am I getting this KeyError in line 42?"
```
